Hotel Rating Application (Spring Boot)

Description

An application for rating hotels, built using Spring Boot microservices architecture.

Microservices
  • Service Registry
  • User Service
  • Rating Service
  • Hotel Service
  • API Gateway
  • Config Server
Technologies
  • Spring Boot
  • Eureka Discovery Client
  • Feign Client
  • Resilience4j (Circuit Breaker & Fault Tolerance)
  • MySQL + PostgreSQL + MongoDB
Project Overview

The Hotel Rating Application is a Spring Boot project that utilizes a microservices architecture for scalability and maintainability. It consists of six microservices responsible for different aspects of the application, including service registration, user management, hotel ratings, and more. The Eureka Discovery Client ensures seamless communication between the microservices, while Feign Client simplifies service-to-service calls. Additionally, Resilience4j is integrated for implementing circuit breaker patterns and fault tolerance, enhancing the application's robustness.
